# Smart Password Analyzer
A web-based password analyzer that checks password strength, detects patterns, and provides suggestions to improve security.

# Features
- Password strength detection (Weak / Medium / Strong)
- Pattern analysis (common, numeric, mixed)
- Suggestions to improve password
- Show/Hide password feature 
- Clean UI with cybersecurity theme

# Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

# How It Works
1. User enters a password
2. System analyzes:
   - Length
   - Uppercase & lowercase letters
   - Numbers & special characters
3. Assigns strength score
4. Detects patterns
5. Displays suggestions
